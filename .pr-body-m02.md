## Cambios

### Backend (commit `1590b54`)
- `src/lib/proveedor-emails.ts`: 4 plantillas HTML + `enviarEmailProveedor` con Resend (degrada si no hay API key).
- `src/lib/proveedores/`:
  - `registro.ts` — `registrarProveedor` transaccional (auth.users + proveedores, compensación si falla insert, reuso de user existente).
  - `actualizar.ts` — update parcial con RLS owner; bloquea campos sensibles.
  - `medios.ts` — upload/delete a bucket `proveedor-medios` con cap `CAP_MEDIOS_FREE`.
  - `admin.ts` — `listarProveedoresAdmin`, `aprobarProveedor`, `suspenderProveedor` + `MOTIVOS_SUSPENSION`.
- **7 API routes**:
  - `POST /api/proveedores/registro` (auto-login + emails welcome + admin).
  - `GET|PATCH /api/proveedores/me`.
  - `POST /api/proveedores/me/medios` + `DELETE /api/proveedores/me/medios/[id]`.
  - `GET /api/admin/proveedores?estado=` (allowlist `getAdminSessionUser`).
  - `POST /api/admin/proveedores/[id]/aprobar` (email al proveedor).
  - `POST /api/admin/proveedores/[id]/suspender` (motivo enumerado).

### UI pública (commit `9acd081`)
- **Landing `/para-proveedores`**: hero, proof points, tabla Free vs Premium, FAQ, CTA dorado sticky.
- **Stepper `/para-proveedores/registro`**: client component `RegistroProveedorFlow` en 3 pasos (Cuenta → Negocio → Visual) + pantalla confirmación, upload 1–6 fotos con preview, draft localStorage TTL 7 días.
- Analytics client-side: `provider_landing_viewed`, `provider_registration_started`, `provider_registration_step_completed` (step + duration_ms), `provider_registration_completed` (category, region, media_count), `provider_media_uploaded`, `provider_registration_abandoned` (sendBeacon → `/api/analytics/beacon`).
- Utility `.input-jurnex` en `globals.css`.

### UI autenticada (commit `d2e4d80`)
- **`/proveedor`**: layout + page con banner de estado (pendiente/aprobado/suspendido con motivo parseado), datos del perfil, galería read-only, aviso M03 para editor.
- **`/admin/proveedores`**: tabs por estado, cards con Aprobar/Suspender, modal con select de motivo + detalle libre; link al nav del admin layout.
